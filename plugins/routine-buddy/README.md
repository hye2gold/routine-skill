# routine-buddy 🫧

직장인을 위한 **루틴 리마인더 플러그인** for Claude Code.

"물 2시간마다 알려줘", "내일 4시 미팅 잊지 마" 처럼 등록해 두면, Claude가 **무슨 얘기를 하고 있든 매 대화마다** 지금 작업 답변과 어우러지게 챙겨줍니다. 알림 앱처럼 따로 보는 게 아니라, 이미 켜둔 Claude Code 대화 속에 녹아드는 게 포인트.

## 어떻게 동작하나

| 부품 | 역할 |
|---|---|
| `/routine` 스킬 | 루틴 등록·관리 (자연어 → 저장소). 특정 날짜는 "그날만? 매일?" 자동 질문 |
| `UserPromptSubmit` 훅 | **매 턴 자동 실행.** 지금 due인 루틴을 계산해 Claude 답변에 자연스럽게 주입 |
| `~/.claude/routine-buddy/routines.json` | 전역 저장소 (모든 세션 공유, 업데이트해도 보존) |

- **주기 루틴**: "물 2시간마다" 등 N분마다 반복. 기본 근무시간(9~19시)에만 알림.
- **작업 세션 루틴**: "Claude Code 3시간째 쓰면 스트레칭" 등 연속 작업 활동 기준.
- **매일 루틴**: "업무일지 매일 18:00" 등 매일 같은 시각.
- **매주 루틴**: "미팅룸 매주 월요일 09:00" 등 요일+시각.
- **특정날 루틴**: "내일 4시 미팅" 등 지정한 날 1회.
- 사용자가 "했어" 하면 완료 처리되어 조용해지고, "이따" 하면 스누즈. 응답 없으면 매 답변마다 작업 흐름에 맞춰 가볍게 다시 상기.
- 초기 온보딩이 끝났고 등록된 루틴도 없으면 완전 무동작(무비용).

## 설치

```
/plugin marketplace add hye2gold/routine-skill
/plugin install routine-buddy@routine-buddy
```

설치 후 Claude Code를 **재시작**하면 훅이 활성화됩니다.

## 첫 실행 온보딩

설치 직후에는 routine-buddy가 아래 순서로 한 번만 초기 세팅을 묻습니다. 사용자가 답할 때까지 매 답변 끝에 계속 묻되, 같은 문장을 그대로 반복하지 않고 매번 표현을 조금 바꿉니다. 여기서 고른 말투는 기본 추천값이며, 실제 리마인더 말투는 새 루틴을 등록할 때마다 따로 저장할 수 있습니다.

1. "설치 완료됐어요! 어떤 말투를 원하나요?"
   - 따뜻하게 챙겨주는 말투(귀여운 이모지 포함)
   - 살짝 쪼아대는 긴급한 말투
   - 차분하고 담백한 말투
   - 직접 원하는 말투를 말해도 됩니다. 말투는 언제든지 바꿀 수 있습니다.
2. "미리 한 번 세팅해두실래요?"
   - 예: 물 마시기, 스트레칭, 업무일지 같은 기본 루틴
3. "취소는 언제든지 할 수 있어요. 확인하셨나요?"
   - 사용자가 예/아니오로 답할 때까지 계속 확인합니다.

### 로컬에서 테스트 설치

```
/plugin marketplace add /path/to/routine-buddy
/plugin install routine-buddy@routine-buddy
```

## 사용

```
/routine 물 2시간마다 알려줘
/routine 스트레칭 매일 오전 10:30 알려줘
/routine 업무일지 매일 18:00 체크해줘
/routine 매주 월요일 오전 9시 미팅룸 잡으라고 알려줘
/routine 내일 오후 4시 팀 미팅 잊지 마        # → "그날만? 매일?" 물어봄
/routine 물은 2시간마다, 스트레칭은 매일 오후 4시에 알려줘
/routine Claude Code 3시간째 쓰고 있으면 스트레칭 하라고 알려줘
/routine 물 알림 취소해줘
```

이후엔 그냥 평소처럼 Claude와 대화하면 됩니다. 때 되면 원래 답변을 먼저 하고, 마지막에 자연스럽게 이어서 챙겨줍니다.

## 동작 규칙

- 오전/오후 없이 "3시", "4시 미팅"처럼 말하면 등록 전에 오전인지 오후인지 먼저 확인합니다.
- 한 번에 여러 루틴을 말하면 각각 독립 루틴으로 나누어 등록합니다.
- 새 루틴을 등록할 때마다 해당 루틴에 적용할 말투를 따로 저장할 수 있습니다.
- "Claude Code 3시간째" 같은 작업 세션 루틴은 앱 실행 시간 자체가 아니라, Claude Code에 프롬프트를 보내며 이어진 작업 흐름을 기준으로 합니다.
- "취소", "그만", "지워"는 해당 루틴을 완전히 삭제합니다. 삭제 후에는 더 이상 알림에 뜨지 않습니다.
- "이따", "잠깐 꺼줘", "30분 뒤에"는 삭제가 아니라 스누즈로 잠시 미룹니다.

## 루틴 등록 방식

| 날짜/시간 구분 | 언제 쓰나요? | 명령 예시 |
|---|---|---|
| 주기형 | N분/N시간마다 반복 | `/routine 물 2시간마다 알려줘` |
| 근무시간 주기형 | 특정 시간대에만 반복 | `/routine 오전 9시부터 오후 7시까지만 물 2시간마다 알려줘` |
| 매일 | 매일 같은 시각 | `/routine 업무일지 매일 오후 6시에 체크해줘` |
| 매주 | 특정 요일마다 반복 | `/routine 매주 월,수,금 오전 10시에 스탠드업 알려줘` |
| 매달 | 매달 특정 날짜 | `/routine 매달 25일 오전 9시에 월급 확인 알려줘` |
| 매년 | 매년 같은 날짜 | `/routine 매년 3월 5일 오전 8시에 기념일 알려줘` |
| 특정날 1회 | 한 번만 있는 일정 | `/routine 내일 오후 4시 팀 미팅 잊지 마` |
| Claude Code 연속 작업 | Claude Code에 계속 프롬프트를 보내며 일한 시간 기준 | `/routine Claude Code 3시간째 쓰고 있으면 스트레칭 하라고 알려줘` |
| 여러 루틴 동시 등록 | 한 문장에 여러 루틴 등록 | `/routine 물은 2시간마다, 스트레칭은 매일 오후 4시에 알려줘` |

오전/오후가 없는 "3시" 같은 표현은 임의로 추측하지 않고 먼저 확인합니다. 특정날 일정도 그날 한 번만인지 반복인지 애매하면 등록 전에 확인합니다.

## 루틴 취소와 상태 변경

| 하고 싶은 일 | 예시 멘트 | 결과 |
|---|---|---|
| 완료 처리 | `했어`, `마셨어`, `스트레칭 했어`, `확인했어` | 이번 차례를 완료하고 다음 주기까지 조용해집니다. |
| 잠깐 미루기 | `30분 뒤에 알려줘`, `이따 알려줘`, `잠깐 꺼줘` | 삭제하지 않고 지정한 시간만큼 스누즈합니다. |
| 완전 삭제 | `물 알림 취소해줘`, `스트레칭 알림 그만`, `미팅 알림 지워줘` | 해당 루틴을 삭제하고 더 이상 띄우지 않습니다. |
| 목록 보기 | `등록된 루틴 보여줘`, `알림 목록 보여줘` | 현재 등록된 루틴과 상태를 보여줍니다. |

## 여러 엣지케이스

| 상황 | 동작 |
|---|---|
| 같은 시간에 루틴이 두 개 이상 겹침 | due인 루틴을 한 번에 모아 Claude 답변 컨텍스트에 넣고, 답변 끝에서 자연스럽게 함께 챙깁니다. |
| Claude가 답변하는 시간과 루틴 시간이 겹침 | 사용자가 메시지를 보낸 시점에 due이면 원래 답변을 먼저 하고 끝에 리마인드합니다. 답변 생성 도중 새로 due가 된 루틴은 다음 사용자 메시지 때 반영됩니다. |
| 등록했던 루틴을 또 등록함 | 자동 병합하지 않고 별도 루틴으로 등록될 수 있습니다. 같은 이름이면 id가 `물-마시기-2`처럼 분리됩니다. 기존 루틴을 바꾸려면 변경 의도를 명확히 말하거나 이전 루틴을 취소한 뒤 다시 등록합니다. |
| 여러 루틴이 동시에 떠 있고 사용자가 "했어"라고만 말함 | 어떤 루틴을 완료한 것인지 분명할 때만 완료 처리하고, 애매하면 먼저 확인합니다. |
| 특정날 1회 알림을 확인함 | `했어` 또는 `확인했어`라고 하면 종료 처리되어 다시 뜨지 않습니다. |
| 특정날 1회 알림을 놓침 | due 후 일정 시간 동안은 계속 상기하고, 너무 오래 지난 일정은 자동으로 조용해집니다. |

## 말투 설정

설치 직후 온보딩에서 기본 말투를 먼저 고릅니다. 이후 새 루틴을 등록할 때마다 해당 루틴에 적용되는 말투를 따로 저장할 수 있습니다. Claude Code의 일반 답변 말투와는 별개입니다.

| 말투 | 예시 멘트 |
|---|---|
| 자연스럽게 | `/routine 루틴 알림은 자연스럽게 말해줘` |
| 따뜻하게 | `/routine 루틴 알림은 챙겨주는 듯한 따뜻한 말투에 귀여운 이모지도 같이 써줘` |
| 긴급하게 | `/routine 루틴 알림은 조금 쪼아대는 긴급한 말투로 해줘` |
| 차분하게 | `/routine 루틴 알림은 차분하고 담백하게 말해줘` |
| 장난스럽게 | `/routine 루틴 알림은 가볍고 장난기 있게 말해줘` |
| 직접 지정 | `/routine 루틴 알림은 친한 동료처럼 짧게 농담 섞어서 챙겨줘` |
| 현재 말투 확인 | `/routine 루틴 알림 말투 뭐야?` |

예를 들어 물 알림은 따뜻하게, 스트레칭 알림은 긴급하게, 업무일지 알림은 차분하게 각각 다르게 설정할 수 있습니다.

## 요구사항

- Python 3 (`python3` PATH에 존재)
- macOS / Linux / WSL

## 저장 위치 / 프라이버시

모든 루틴은 로컬 `~/.claude/routine-buddy/` 에 저장됩니다.
환경변수 `ROUTINE_BUDDY_HOME` 으로 저장 위치를 바꿀 수 있습니다.

due인 루틴의 label/id는 Claude가 답변 속에서 자연스럽게 챙길 수 있도록 해당 턴의 대화 컨텍스트에 포함됩니다.
